Nós do Compare BSB coletamos e tratamos a menor quantidade de dados o possível para manter as funcionalidades da aplicação.

Os Dados Coletados são, Localização, Captura de imagem (QR Code), email e nome.

Nós tratamos os dados dos mercados e produtos que são retornados pelo QR Code lido.

A localização será utilizada apenas para melhorar os resultados das buscas obtidas na aplicação.
